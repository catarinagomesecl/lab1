Programacao na Web

Na Escola do Comercio de Lisboa, na disciplina de Tecnologia 1 no 3ºTIIGR, iremos aprender Programação WEB 😅 e abordaremos os principios básicos associados ao desenvolvimento para a Web, apenas do lado front-end, lado visual. Tambem sabemos que existe um lado back-end, que e o lado do servidor, mas nao iremos abordar esse lado

Aprender-se-ao as seguintes tecnologias 🌐:

  • HTML
  • CSS
  • JavaScript

De forma a consolidar todos os conhecimentos, serao feitos alguns laboratorios.
